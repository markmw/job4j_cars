#job4j_cars

Проект "АвтоМаг".

Сайт по продаже машин.

Используется:
1. JDK17
2. Maven
3. Spring boot 2.5.2
4. Thymeleaf.
5. HTML5/Bootstrap.
6. Liquibase 3.6.2
7. Hibernate.

Основные возможности:
1. Добавление объявлений о продаже автомобиля.
2. Изменение статуса объявления.
3. Просмотр объявлений и поиск с помощью фильтров(Criteria_API).
4. Регистрация и вход.
